qmap

QMap is a simple data collection program built using QGIS and Python. The program was built by Southern Downs Regional Council in order to aid in field data collection

At the current time it is implmented as a QGIS Python plugin and is aimed at Windows users, however the intention is to change it into a standalone QGIS program and run on any platform. The program is only in early alpha stage but is functional.

Loading screen presents users with a simple list of projects to select from.

The large buttons with text aid in usability on a pen or touch based device.

An example daily water job sheet with all the information that can be collected at one job site. Forms can be built using Qt Designer and fields are filled using the name of the control, saving on configuration. Forms can be made to run full screen on the device or in a floating dialog depending on the need.

Freehand drawings can be completed and attached to records.
