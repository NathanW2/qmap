=================================
Downloading and Installing QGIS
=================================

.. toctree::
   :maxdepth: 2
   :numbered:

The following steps are needed to download install QGIS and any required dependencies.

1. Download the OSGeo4W installer from the OSGeo4W site	
2. Run the installer and select :guilabel:`Advanced Install`

.. note:: If running on Windows Vista or above run the installer as an administrator.

3. Continue until you are given the tree view of different packages
4. Select :menuselection:`desktop --> qgis-dev`
5. Select :menuselection:`libs --> qt4-deval`
6. Continue with install
